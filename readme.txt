Кроссплатформенная библиотека для детекта движения с помощью веб камеры


+

19.04
На данный момент имеется кросплатформенная программа которая может захватывать с веб камеры
видео поток и детектировать движение в нем.
Программа реализована с помощью библиотеки OpenCV (Open Source Computer Vision Library, библиотека компьютерного зрения с открытым исходным кодом)
